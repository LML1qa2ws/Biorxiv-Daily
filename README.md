# Biorxiv-Daily
科学问题跟进，文献学习及分享。
生命科学领域预印本论文的定期追踪与精读汇总，覆盖 bioRxiv / medRxiv 最新动态。
## 内容结构
```
Daily/   每日动态报（HTML）—— 精选当日重要预印本，按类别归档
```
## 报告分类
每篇日报将收录论文划分为三类：
| 类别 | 说明 |
|------|------|
| 综述类 | 领域综述与系统性回顾 |
| 研究类 | 实验研究与机制探究 |
| 技术类 | 计算方法、工具与平台 |
## 主要涉及领域
- 肠道菌群与微生物代谢组
- 肿瘤免疫与免疫治疗
- 空间转录组与单细胞组学
- 表观遗传与基因组调控
- 神经科学与脑疾病
- 生物信息学工具与算法
## 文件命名规则
- 日报：`预印本科研动态日报-YYYY-MM-DD.html`
